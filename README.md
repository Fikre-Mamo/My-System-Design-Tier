# System Design Tier List

A sleek, interactive web application that allows users to categorize various system design concepts into a tier list (S, A, B, C, D).

## Features

- **Interactive Drag & Drop:** Easily move system design concepts between different tiers.
- **Touch Support:** Fully functional on mobile devices with touch drag and drop.
- **Detailed Concept Panels:** Click on any concept to open a side panel (or bottom sheet on mobile) containing details about the concept, when it is useful, and common tools associated with it.
- **Export Feature:** Download your completed tier list as a PNG image.
- **Responsive Design:** A beautiful, responsive interface built with Tailwind CSS.

## How to Run

Since this is a static web application, you can simply open the `index.html` file in any modern web browser to start using it. The project is organized into `index.html`, `style.css`, and `script.js` for modularity.
